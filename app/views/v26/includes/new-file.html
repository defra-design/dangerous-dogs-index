Manage files for application pack emails and letters

Test and replace the files attached to application pack emails and sent in the post.

To test a file, select ‘Test the file’.

To use a new file:

  1. Name the new file differently from the current live version.
  2. Upload the file.
  3. Test it.
  4. Make it live.
  5. Delete the old version.

Only 1 file to be emailed and 1 file to be posted can be live at a time.
