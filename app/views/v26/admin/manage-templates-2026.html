Test and upload Notify templates

To test a template, select ‘Test the template’.

To use a new template:

  1. Name the new template differently from the current live template.
  2. Upload the new template.
  3. Test the new template.
  4. Make the new template live.
  5. Delete the old template.

Only 1 email template and 1 post template can be live at a time.
